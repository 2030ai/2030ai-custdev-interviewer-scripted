# Персона 6: Алексей Новиков

**Возраст:** 28 лет
**Должность:** Project Manager в digital-агентстве
**Опыт:** 3 года в проектном управлении
**Зарплата:** 180,000₽/месяц
**Город:** Екатеринбург

## Контекст

Ведёт одновременно 5-7 клиентских проектов. Каждый день — созвоны с разными клиентами, разными командами. 15-20 встреч в неделю, и каждая — с другим контекстом.

Главная боль — клиенты меняют требования и потом отрицают это. Ведёт заметки в Google Docs, но на это уходит много времени, и всё равно иногда что-то упускает.

## Боли

- Клиенты регулярно говорят "мы так не договаривались", хотя договаривались
- Переключение между проектами требует времени на "вспомнить контекст"
- При передаче проекта коллеге уходит 2-3 дня на введение в курс дела
- Нет времени на детальные заметки — проще созвониться ещё раз

## Цитата

"Каждый второй клиент через месяц говорит 'мы такого не просили'. А у меня нет записи с точными словами — только мои заметки, которые они не признают."

## Технический стек встреч

- Google Meet / Zoom (зависит от клиента)
- Google Docs для заметок
- Asana для задач
- Telegram для коммуникации с клиентами
