# CustDev Interviewer

Автоматизированная валидация стартап-идей через симулированные custdev-интервью с помощью AI.

## Что вы получите

За 5-10 минут работы с AI вы получите:
- 10 детальных custdev-интервью с вашей целевой аудиторией
- Аналитику результатов с таблицей ответов
- 3 причины ЗАПУСКАТЬ и 3 причины НЕ ЗАПУСКАТЬ стартап
- Итоговую рекомендацию по запуску
- 3 готовых сценария продающих роликов

## Как запустить

### Шаг 1: Откройте Claude Code

Убедитесь, что у вас установлен [Claude Code](https://claude.ai/claude-code)

### Шаг 2: Сформулируйте идею

Подготовьте краткое описание вашей стартап-идеи. Например:
- "Uber для вызова дронов-доставщиков"
- "Школа AI курсов для бухгалтеров"
- "Маркетплейс услуг психологов для подростков"

### Шаг 3: Запустите проект

В Claude Code напишите:

```
@agents.md я хочу проверить идею стартапа: [ваша идея]
```

**Пример:**
```
@agents.md я хочу проверить идею стартапа: Uber для вызова дронов-доставщиков
```

### Шаг 4: Дождитесь результатов

Claude Code автоматически:
1. Структурирует вашу идею (Аудитория, Проблема, Решение)
2. Сгенерирует 10 вопросов для интервью
3. Создаст 10 уникальных персон из целевой аудитории
4. Проведет 10 детальных интервью
5. Проанализирует результаты
6. Создаст сценарии продающих роликов

Все результаты будут сохранены в папке `outputs/ваша-идея/`

## Структура результатов

После завершения работы вы найдете:

```
outputs/ваша-идея/
├── 1-startup-mix.md       # Структурированное описание стартапа
├── 3-questions.md         # 10 вопросов для интервью
├── 4-person-01.md         # Персона 1
├── 4-person-02.md         # Персона 2
├── ...
├── 4-person-10.md         # Персона 10
├── 5-script-01.md         # Интервью с персоной 1
├── 5-script-02.md         # Интервью с персоной 2
├── ...
├── 5-script-10.md         # Интервью с персоной 10
├── 6-analyse.md           # 📊 ГЛАВНЫЙ ФАЙЛ: Анализ всех интервью
└── 7-reels.md             # 🎬 Сценарии продающих роликов
```

## Основные файлы результатов

### 📊 6-analyse.md
Содержит:
- Таблицу частых ответов по каждому вопросу
- 3 причины ЗАПУСКАТЬ стартап
- 3 причины НЕ ЗАПУСКАТЬ стартап
- Итоговую рекомендацию с конкретными шагами

### 🎬 7-reels.md
Содержит:
- 3 готовых сценария для видео-роликов
- Тайминг и визуальное описание каждого кадра
- Тексты для закадрового голоса

## Советы для лучших результатов

1. **Будьте конкретны** в описании идеи
   - ❌ Плохо: "Приложение для здоровья"
   - ✅ Хорошо: "Приложение для отслеживания водного баланса для офисных работников"

2. **Укажите целевую аудиторию**, если она специфична
   - "Маркетплейс handmade украшений для девушек 18-25 лет"

3. **Опишите ключевую механику**, если она важна
   - "Подписка на кофе с доставкой на работу по расписанию"
